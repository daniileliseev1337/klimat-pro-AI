# Дизайн-спек: UI-редизайн дашборда КЛИМАТ-ПРО (max-wow black & gold)

> Дата согласования: 2026-06-23. Статус: **дизайн утверждён владельцем** (эталон одобрен).
> Источник истины по стилю — живой эталон-мокап (см. ниже). Этот документ фиксирует РЕШЕНИЯ;
> точные значения/код — в эталоне.

## Проблема

Прошлый редизайн (premium-dark, коммиты `2353d52`..`3db9882`) оптимизировал под «сдержанный
тихий инструмент»: фон — одно еле заметное золотое пятно (`opacity 0.10`, дрейф 26с) + статичные
точки. Владелец же хотел **современный, инновационный, живой** вид. Итог — на проде видны только
hover-переливания, «наобещали, а по факту переливания». Это не баг, а промах в подаче/амбиции.

## Согласованное направление

Палитра остаётся **black & gold** (бренд КЛИМАТ-ПРО), но всё оживляется до **максимального wow**.
Весь «цвет» из других вариантов (cyan/violet/перламутр) переведён в **золото** — монохромный premium.

### Решения (выбор владельца)

| Аспект | Решение |
|---|---|
| Палитра | black & gold, переменные из `src/index.css` (`--gold #d4af37`, `--gold-bright #f0d878`, `--gold-deep #9c7c22`, `--bg #0a0a0a`) |
| Живой фон | **Комбо** на canvas: золотой aurora (дрейфующие пятна) + сеть-созвездие (узлы+линии с бегущими импульсами) + плавающие золотые искры. Один глобальный canvas. |
| Логотип — символ | значок «потоки воздуха» (как mockup-1) + **breathing glow** (медленная пульсация свечения ~4.6с) |
| Логотип — текст | «КЛИМАТ-ПРО» с золотым **shimmer**-переливом |
| Карточки — рамка | золотая «слиток»-рамка: градиент-бордер через `mask-composite` + бегущий блеск |
| Карточки — движение | **3D-tilt + золотой spotlight ТОЛЬКО на крупных карточках** (KPI, секции). Строки списков — лёгкая золотая подсветка без 3D (производительность) |
| Профиль | имя + должность («Даниил К. / Гл. инженер») |
| Cmd+K | компактный **видимый бейдж ⌘K** в шапке (вспышка по нажатию) |
| Доп. фишки | count-up цифр (0→N), magnetic CTA, stagger-reveal появления |
| Верхняя панель | без жёсткой границы; низ растворяется в прозрачность, бока «затушёваны» маской («туман», не плашка с углами) |
| Доступность | уважать `prefers-reduced-motion` (глушить декор-анимации); desktop-first (tilt без тача) |

## Технические заметки и грабли (важно для реализации)

1. **Рывок анимаций** — однонаправленный `linear infinite` на `background-position` даёт скачок
   в точке перезапуска цикла. Решение: `ease-in-out infinite alternate` (блик «качается», без разрыва).
   Применено к shimmer текста и ingot-рамке.
2. **Защита от «пустого дашборда»** — каждая JS-подсистема в `try/catch`; глобальный таймер 2.5с
   принудительно показывает `.reveal`-контент, даже если что-то упало. (Урок mockup-2.)
3. **Read-only присваивание** — НЕ присваивать `canvas.clientWidth/clientHeight` (read-only,
   в strict-режиме = TypeError, рушит весь скрипт). Использовать `W = window.innerWidth`. (Root cause
   бага «пустой mockup-2».)
4. **Производительность** — tilt только на крупных карточках; один canvas-фон; rAF; reduced-motion
   рисует один статичный кадр.
5. **Среда** (см. CLAUDE.md / память проекта): F: сбоит на fsync больших файлов — App.jsx править
   через C:\temp → copy на F. git только Windows-сторона. localhost/file:// для визуальной проверки
   недоступны (corp-прокси + playwright блок) — визуальную приёмку делает владелец.
