# Дизайн-полировка klimat-pro-AI — спека фазы (2026-06-24)

Сбор фидбэка владельца + идеи + план. Premium-dark уже внедрён (карточки/шапка —
«конфетка»); эта фаза — добить непроработанные зоны и мелочи. Перед реализацией:
brainstorming + frontend-design, по приоритету.

## Принцип работы фазы (ВАЖНО)
**Claude Design — основной инструмент дизайна** (round-trip: правим облик в облаке с
живым контролем владельца → возвращаем в код). Это в том числе **боевое тестирование**
инструмента (как договаривались по реворку базы), а не «отдать для галочки».
**Claude Code (Sonnet/Opus) — конвейер ниже:** перенос утверждённого дизайна в реальный
`App.jsx`/`index.css`, build + deploy, БД/логика. Не «код важнее» — разделение труда.

- Через Claude Design: логотипы (LogoMark уже запушен), золотые акценты, типографика/
  начертание, варианты liquid glass (приблизительно — фон/контент не настоящие).
- В коде: адаптив/мобайл, структура шапки, модалки, max-width, история-иконка, деплой.

## Задачи (из разбора фидбэка)

### 📱 Мобайл (приоритет — болит)
- M-1. Всё ужато/скомкано, тесно — пройтись по spacing.
- M-2. **Золото доминирует над чёрным** на малых экранах — сместить баланс (меньше золотых
  плоскостей, больше золотых hairline-линий).
- M-3. Шапка с лого **прижата влево** — распределить/центрировать.

### 🖥 ПК
- D-1. Незанятое место на крупных мониторах + местами **пустоты** → max-width контейнер,
  баланс плотности.
- D-2. **Чёрная полоса под прозрачной шапкой** (зона вкладок Дашборд/Проекты/ИД) портит вид.
  Связано с `backdrop-filter` (аудит M1). Решение: зона вкладок прозрачная + золотой hairline
  под активной вкладкой вместо тёмной плашки.

### ✨ Акценты
- A-1. **Тонкие золотые линии** — hairline 1px, золото ~10–15% opacity: разделители секций,
  подчёркивание активной вкладки (закрывает D-2), сепаратор под шапкой. Золото как волосок,
  не заливка (помогает и M-2).
- A-2. **Рукописный шрифт (script) для одного из двух слоганов шапки.** НЕ менять слова.
  Кандидаты: «климат, рассчитанный профессионально» (`brand-static-gold`) ИЛИ
  «Заботимся о тех, кто внутри». Рекомендация: рукописным — «Заботимся о тех, кто внутри»
  (тёплый, как подпись). Примерить оба. Дозировка: ровно одно место, не в рабочей зоне.
- A-3. **Дата** — `tabular-nums`, приглушённый `--text-tertiary`, опц. золотая точка-сепаратор.
  Уточнить конкретное место, где «не к месту».
- A-4. **Зелёный цвет профиля → золото/нейтраль** (зелёный чужой в black&gold).
